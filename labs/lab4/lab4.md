# Лабораторная работа №4 по предмету "Операционные системы". Преподаватель Данилов А.С.

## Задание
- лабораторная очень простая, чтобы все успели
- задание выполняется в линукс
- если вам непонятно, можете начать с этого видео https://www.youtube.com/watch?v=dR4DIo7HYoU (но, возможно, вы найдете и другие более удобные и понятные для вас материалы)
- создать три пользователя alice, bob, eve
- alice и bob находятся в группе developers, eve нет
- выдать alice sudo права, остальным нет
- создать папку project, внутри этой папки папки src, logs, public, tmp
- владелец project alice группа developers
- в папке src читать могут все, писать (w) только владелец (alice) и группа developers
- к папке logs имеют доступ только alice и группа developers, у остальных никакого доступа нет
- сделайте так, что в папке tmp у каждого есть право на запись, но удалить можно только свои файлы (sticky bit)
- сделайте пару хардлинков и пару симлинков
- во время сдачи надо будет быть готовым показать и объяснить, что вы сделали (то есть уметь показать пользователей, группы, права доступа и прочее)

## Отчет
- титульный лист
- постановка задания
- процесс решения (команды)
- полученные результаты (парочку скринов, что вы все сделали правильно)
- выводы (напишите просто, что было сделано)
